# 约会预算规划 & 首次见面后勤指南

> 从实际约会经验中提炼的后勤规划方法论。适用于刚毕业/学生/经济紧张的用户。

## 核心原则

1. **预算倒推，不是正推** — 先问预算，再在预算内安排最优方案
2. **省钱不丢人，心态稳才是核心** — 1000块从容约会 > 2000块紧张算计
3. **第一印象靠收拾，不靠花钱** — 发型、穿搭、干净 > 高端场所
4. **轻装上阵** — 第一次见面背书包/拖行李箱 = 画面感崩塌

## 交通优化

| 方式 | 单程成本 | 适合场景 |
|------|---------|---------|
| 高铁二等座 | 300-400 | 时间紧、预算充足 |
| 普通火车硬座 | 100-130 | 预算紧张、能接受6-12h |
| 普通火车硬卧 | 150-200 | 夜车睡一觉到，省住宿 |
| 高铁去+火车回 | 折中 | 去时精神好，回时省钱 |

**夜车策略**：晚上出发的火车 = 省一晚住宿 + 早上到有时间打扮。硬卧是性价比之王。

## 住宿优化

| 类型 | 价格 | 适合场景 |
|------|------|---------|
| 青旅床位 | 40-60 | 最省钱 |
| 快捷酒店（7天/如家/汉庭） | 80-130 | 干净有保障，可寄存行李 |
| 钟点房 | 50-80/4h | 只需洗漱整理，不住宿 |

**行李寄存技巧**：酒店退房后行李可免费寄前台，轻装赴约。地铁站/商场储物柜（5-15元/次）是备选。

## 首次见面约会流程（预算500-700元版）

### 核心思路：免费场景为主 + 1-2次小消费

```
见面 → 免费场景建立舒适感 → 低价消费场景过渡 → 免费场景推进关系 → 结束
```

### 免费/低成本约会场景库

| 场景 | 花费 | 适合阶段 | 备注 |
|------|------|---------|------|
| 商场逛逛 | 0 | 建立舒适感 | 天然话题多，空调舒适 |
| 公园/湖边散步 | 0 | 推进关系 | 夜景+散步=天然浪漫 |
| 老街/文创街区 | 0 | 打破舒适感 | 拍照出片，发现感强 |
| 博物馆/美术馆 | 0 | 建立舒适感 | 显格调，免费预约 |
| 奶茶/咖啡 | 15-30/杯 | 全程可用 | 人手一杯，自然不刻意 |
| 人均50餐厅 | 100-120/两人 | 正餐 | 商场连锁（绿茶、曼玉等） |

### 转场节奏

**至少转2次场**，每次转场 = 关系升级信号：
1. 商场/咖啡馆（舒适）→ 老街/公园散步（打破舒适）
2. 散步 → 吃饭（回到舒适）→ 夜景散步（再次打破）

## 常见后勤问题

### 背书包约会怎么办？
- **最优**：酒店寄存，轻装赴约
- **次选**：地铁站/商场储物柜（5-15元）
- **底线**：如果必须带，选好看的双肩包，不要太鼓

### 要不要帮对方买车票？
- **不要主动买** — 框架D：让她先投资小成本（坐车来找你本身就是投资）
- **可以请吃饭** — 但不要全程抢着买单，让她买个奶茶

### 第一次见面穿搭
- 干净清爽 > 花哨（框架B：建模是入场券）
- 提前洗头、夹头发、涂素颜霜（183身高+收拾 = 足够）
- 不需要新衣服，干净整洁就行

## 多方案对比方法论

| 维度 | 权重 | 方案A | 方案B | 方案C |
|------|------|-------|-------|-------|
| 约会时间充裕度 | 高 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| 打扮准备时间 | 高 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 轻装程度 | 中 | ⭐⭐⭐ | ⭐ | ⭐⭐⭐ |
| 约会从容度 | 高 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| 请假成本 | 中 | ⭐ | ⭐⭐⭐ | ⭐⭐ |

选**加权得分最高**的方案，而不是最便宜或最方便的。
