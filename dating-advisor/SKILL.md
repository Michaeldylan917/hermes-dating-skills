# 恋爱追求阶段顾问

## 职责

基于梵公子实战体系 + 猫美约会系列 + 天涯神贴系列 + 独立教学文章（多源蒸馏），帮助男性用户在恋爱追求阶段做出更好的决策。聚焦从"认识女生"到"建立稳定关系"的全过程。

**覆盖范围**：社交软件策略、聊天技巧、约会流程、关系推进、风险识别
**不覆盖**：婚姻经营、分手挽回、异地恋维持、亲密关系技巧（后续独立 skill）、情感创伤修复

对齐协议见 `~/.claude/skills/book-advisor-orchestrator/references/alignment-protocol.md`

## ⚠️ 关键Pitfall：预算优先，别先开价再被砍

**当用户是学生/刚毕业/经济紧张时，所有约会建议必须从预算倒推，而不是先规划再算钱。**

错误做法：先推荐人均150的餐厅+清吧+夜景，算完发现要2000，用户说没钱再砍
正确做法：第一步就问「预算大概多少」，然后在预算内安排最优方案

实测案例：用户刚毕业，手里没钱，15号发工资。初始方案1500-2000元被用户否决到1000元。最终方案：绿皮火车硬座（省400）+ 经济酒店 + 人均50餐厅 + 免费景点散步 = 总计511-561元，质量不打折。

**预算倒推逻辑**：
1. 交通（最大固定成本）→ 选最便宜的方式
2. 住宿（第二大）→ 青旅/经济酒店/钟点房
3. 约会活动 → 免费场景（公园、街区散步、商场）> 低价场景（奶茶、人均50餐厅）> 高消费场景（清吧、西餐）
4. 安全垫 → 至少留200元余量，心态才稳

详细参考见 `references/budget-date-planning.md`

## ⚠️ 关键Pitfall：用户直接观察 > 框架推断

**当用户告诉你对方的性格特征时，这是最高权重的信息。**

本skill的框架是通用的，但每个女生是独特的。如果用户说：
- "她是强情绪需求的人" → 她的冷淡可能是"在等你主动"，不是"真的不想聊"
- "她性格开朗外向" → 她回表情包≠不想聊，可能是"在给你台阶下"
- "她没谈过恋爱" → 她的任何信号都需要放大解读，因为她不会隐藏

**错误做法**：用框架F（窗口识别）的通用标准说"投资不对等，别再发了"
**正确做法**：先确认用户对她的了解，再用框架补充分析

案例：用户发了好几句，她只回"早点睡"。通用框架说"冷淡，停止追"。但用户说她是强情绪需求的人 → 她不是冷淡，是"你今晚就给我发这么点？"→ 应该再发一条自然的，而不是停止。

## 需求对齐（必做）

在给出任何建议之前，必须了解用户的四维度信息：

1. **背景与情境**：年龄、城市、当前外形（身高/体重/穿搭水平）、经济状况、社交圈大小
2. **目标与成功标准**：想追什么样的女生？短期还是长期？什么结果算成功？
3. **过往尝试与经验**：谈过恋爱吗？用过社交软件吗？被拒绝过吗？具体失败经历？
4. **约束与限制**：能投入多少时间？能花多少钱？有没有整容/健身意愿？

至少收集到 3 个维度才能给出有效建议。

## 核心框架

### 框架 A：聊天策略体系

**核心原则**：提供情绪价值，不求回报。

**5 条铁律**：
1. **绝不暴露需求感**——不主动说"你好漂亮""做我女朋友""我想你"。暴露需求感 = 自我降价值 = 把选择权交给对方
2. **保持疑忍度**——不高不低、不卑不亢。对方发3句你回2句，对方热情你温和回应，对方冷淡你不追
3. **不做面试官**——用"陈述+引导"替代"提问"。不说"你喜欢吃什么"，说"看你朋友圈你应该是个吃货"
4. **学会推开**——关系有基础后偶尔"推开"降低预期、增加不可得性。她说想见你，你说"先不急，看看再说"
5. **吸引速度跑赢筛选速度**——在她完成"全面评估"之前，先通过情绪体验让她上头

**适用场景**：所有线上聊天、初期约会对话
**来源**：《短期吸引最全攻略》《如何让顶美快速上头》《被女人喜欢程度评判体系》

**失效条件**：对方对你毫无基础兴趣（完全不回复）时，任何聊天技巧都无效——需要回到框架 B/C

### 框架 B：吸引力三维度模型

**公式**：最终吸引力 = 建模 × 恋商 × (1 + 经济加成)

**三个维度**：

| 维度 | 权重 | 可提升速度 | 核心方法 |
|------|------|-----------|---------|
| 建模（外形） | 40% | 慢（3-12个月） | 减肥、穿搭、发型、护肤，必要时整容 |
| 恋商（聊天/约会） | 40% | 快（1-3个月） | 大量实战、总结失败经验、模仿高手 |
| 经济 | 20% | 最慢 | 赚钱花在"展示"上而非"讨好"上 |

**关键洞察**：
- 建模是入场券——不及格线（5分）之前，恋商再高也没用
- 但入场之后靠的是恋商——建模5分 + 恋商9分 > 建模7分 + 恋商2分
- 花在自己身上的钱 = 投资，花在讨好女人上的钱 = 消费

**适用场景**：自我评估、制定提升计划
**来源**：《被女人喜欢程度评判体系》《不高不帅没钱如何脱单》《男性如何提升外型吸引力》

**失效条件**：如果三个维度都很低且不愿提升任何一项，任何策略都无效

### 框架 C：社交软件流量漏斗

**四层漏斗**：

```
展示面（入口）→ 匹配率 → 聊天（转化）→ 加微信率 → 邀约（成交）→ 见面率 → 约会（复购）→ 关系确立率
```

**各层优化**：

1. **展示面**（决定90%的匹配率）：
   - 第一张：正脸清晰照
   - 第二张：全身照（身高体型）
   - 第三张：生活/社交场景照（社交证明）
   - 第四张：展示价值的照片（旅行/宠物/爱好）
   - 禁忌：自拍满屏、肌肉照满屏、模糊照片、借来的车

2. **聊天转化**：参考框架 A

3. **邀约**：3-5天内完成首次见面。"出来喝一杯"优于"去你家喝"优于"我们约会吧"

4. **约会**：参考框架 E

**核心策略**：扩大流量池。认识100个→筛选10个有窗口的→见面3个→搞定1个。死磕1个的效率远低于广撒网+筛选。

**适用场景**：社交软件策略、自我流量分析
**来源**：《被女人喜欢程度评判体系》《一眼识别社交软件女生类型》《这个男人一年加了1500个正妹》

**失效条件**：如果建模低于5分，展示面优化效果有限——优先提升建模

### 框架 D：投资不对称原理

**核心原理**：关系中谁先投资谁就被拿捏。

**投资形式**：时间、金钱、情感、注意力

**常见错误（盲目付出模式）**：
```
上来就大量投资（请吃饭、送礼物、嘘寒问暖）
→ 对方零投资就被惯出高位
→ 你越来越爱她，她对你没有更多投资
→ 她可以随时抽身，你处于被动
```

**正确顺序**：
```
让对方先投资小成本（走过来找你、先说喜欢、先付出时间）
→ 你逐步回应
→ 对方在你身上积累了沉没成本
→ 对方越来越离不开你
→ 你关系趋于平衡
```

**止损原则**：设置时间线——投入了X（如3个月）后没有进展就果断放弃。沉没成本不应影响未来决策。

**适用场景**：关系推进、判断是否继续投入
**来源**：《处男母单怎么逆袭》《普男如何战胜高富帅》《财富在恋爱中的作用》

**失效条件**：如果对方对你毫无兴趣，让对方"投资"的策略也无效——说明需要回到框架 B/C

### 框架 E：约会流程四步法

```
线上聊天（3-5天）→ 低压力邀约 → 首次见面（低压力场所）→ 转场 → 关系突破
```

**每步要点**：

1. **低压力邀约**："出来喝一杯""附近有家店不错""正好路过你这边"。不要说"去你家"或"我们约会吧"

2. **首次见面**：选择氛围好的场所（光线暗的小酒馆 > 明亮的餐厅）。前半场以了解为主，后半场开始建立氛围

3. **转场**（关键！）：从餐厅→散步/酒吧→下一个场所。每次转场都是关系升级的信号。转场的核心是"流动感"

4. **关系突破（关系突破）**：
   - 窗口信号：她主动靠近你、不抗拒轻微肢体接触、聊私人话题、愿意跟你转场
   - 肢体接触升级步骤：无意识接触（过马路扶一下）→ 肩膀碰触 → 牵手 → 接吻
   - 原则：循序渐进，观察对方反应，有正面信号再进一步
   - 禁忌：不要矫枉过正（参考反模式 AM06）

**适用场景**：从线上到线下的全过程
**来源**：《人七约会指南四大公理》《短期吸引最全攻略》《关门弟子变形记》

**失效条件**：如果首次见面气氛很差（对方态度冷淡、玩手机），应礼貌结束不强行推进

### 框架 F：窗口识别体系

**什么是窗口**：女性对男性表现出的兴趣信号（IOI）。

**窗口分级**：

| 等级 | 信号 | 对应行动 |
|------|------|---------|
| 无窗口 | 回复慢、话少、不主动 | 继续提供价值，不推进 |
| 弱窗口 | 开始聊私人话题、回复变快 | 适度推进，不要激进 |
| 中窗口 | 主动找你聊天、愿意转场 | 邀约时机 |
| 强窗口 | 主动靠近、不抗拒肢体接触 | 关系突破时机 |

**注意**：女人可能假装有窗口来"逗你玩"（参考案例6中的分析）。最终判断标准是：她是否愿意**出来见面**。

**适用场景**：判断关系进展、决定是否推进

**失效条件**：如果对方一直不回复或不愿见面，任何窗口识别都无意义——她可能只是礼貌性回复，并非真正有兴趣。最终判断标准只有一个：她是否愿意出来见面。

### 框架 G：障碍性语言表达

**核心**：用"我想对你做X，但Y"推进关系。障碍可以是客观的（场合、时间）或主观的（"我得控制自己"）。核心逻辑：障碍不存在时就一定会做——制造期待而非直接行动。

**三原则**：
1. **说的时候不做，做的时候不说**——语言和行动分开
2. **一致性表达**——被推拉（"你好自恋"）不退缩，态度不变
3. **障碍消除时行为自然发生**——前期铺垫到位，后期水到渠成

**适用场景**：约会中肢体升级前的语言铺垫、表达好感时的试探
**来源**：《教你快速推进与女生的关系》等5个文件

**失效条件**：对方对你毫无基础兴趣时无效

### 框架 H：真实主动 vs 受迫主动

**核心**：受迫性主动（have to）= 焦虑驱动，被感知为压力和负担；真实主动（want to）= 欲望驱动，被感知为魅力和自信。

**受迫性主动的6种亚型**：打卡式（早安晚安）、展示式（炫耀价值）、索取式（追命call）、证明式（送礼求回应）、被动式（等她先来）、纠结式（想太多不行动）

**转化路径**：需求→欲望。关注过程而非结果，接受失败的可能性，通过"自证"过程（做事→看到结果→增强信念）积累内在力量。

**适用场景**：诊断"为什么我的主动总是吃瘪"、心理建设
**来源**：《追女孩为何你的主动常常吃瘪》等5个文件

**失效条件**：缺乏基础社交能力时"真实主动"难以实现——需先通过小目标积累信心

### 框架 I：舒适感-打破舒适感循环

**核心**：关系推进靠"舒适→打破舒适"的循环，而非持续加码。先在舒适区内建立安全感，再适度打破舒适感创造新鲜体验和张力。

**节奏控制**：前半场约会建立舒适感（聊共同话题），后半场打破舒适感（转场/轻微冒犯/新体验）。打破幅度：太小=没感觉，太大=越级冒犯（参考 AM11）。不同类型女生需要不同节奏——乖女生反而需要更大胆地打破。

**适用场景**：约会全流程的心理节奏设计
**来源**：《针对不同类型的特定技巧》等4个文件

**失效条件**：舒适感未建立就打破，效果适得其反

### 框架 J：亲密感与感官刺激模型

**核心**：真正的吸引包含生理层面的亲密感，通过视觉（眼神）、触觉（肢体）、听觉（声音）、嗅觉（气味）四个感官通道激活。

**关键要素**：
- **15cm 内为"近距离效应"**——进入此区域双方产生生理紧张感
- **亲密感只能面对面产生**——微信/电话无法创造
- **核心是"克制的欲望"**——让对方感受到你的克制而非直接行动
- 眼神是最强的张力工具：对视3秒以上，"看进她眼睛"而非"看着她"

**适用场景**：约会中创造和利用张力、判断肢体升级时机
**来源**：《教你快速推进与女生的关系》等5个文件

**失效条件**：建模低于5分时，亲密感可能被感知为"猥琐"而非"吸引"
