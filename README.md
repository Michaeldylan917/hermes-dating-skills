# Hermes Dating Skills

4 open-source [Hermes Agent](https://hermes-agent.nousresearch.com) skills for dating and relationships.

> Disclaimer: These skills are for personal development and understanding relationship dynamics. They are based on publicly available materials and should be used ethically and respectfully.

## Skills Overview

| Skill | Source | Purpose |
|-------|--------|---------|
| dating-advisor | 梵公子实战体系 + 猫美/天涯多源蒸馏 | 聊天策略、窗口识别、投资不对称、约会流程、反模式诊断 |
| women-psychology-advisor | 渡边淳一《女人这东西》 | 从生物学/荷尔蒙视角理解女性行为 |
| tong-jincheng-perspective | 童锦程视角 | 执行层话术、框架主导、价值交换 |
| attract-advisor | 莉尔·朗兹《如何让你爱的人爱上你》 | 科学研究 + 实战，吸引力建设、全流程策略 |
